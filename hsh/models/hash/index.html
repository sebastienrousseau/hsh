Core hash::Hash type. Core Hash type for storing and verifying password hashes.

Built on the RustCrypto stack (Phase 1):

  • Argon2 family via the [argon2] crate, with Argon2id as the recommended default per RFC 9106 §4.
  • Bcrypt with the explicit 72-byte safety rail (CVE-2025-22228 class).
  • Scrypt with configurable parameters; default is the OWASP-2025 minimum (N = 2^17).
  • Salts generated from getrandom (OS CSPRNG).
  • Verification is constant-time everywhere via subtle.
  • Secret material is zeroed on drop via [zeroize].
