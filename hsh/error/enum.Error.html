pub enum Error {

    Decode(DecodeError),
    Pepper(Cow<'static, str>),
}
Expand description

The error type returned by all fallible hsh operations.

Variants (Non-exhaustive)§

This enum is marked as non-exhaustive
Non-exhaustive enums could have additional variants added in future. Therefore, when matching against variants of non-exhaustive enums, an extra wildcard arm must be added to account for any future variants.
§

UnsupportedAlgorithm(Cow<'static, str>)

The requested algorithm string did not match any supported variant.

§

InvalidHashString(Cow<'static, str>)

The PHC / serialized hash string could not be parsed.

§

InvalidParameter(Cow<'static, str>)

A supplied parameter (cost, memory, iterations…) was outside the algorithm’s valid range.

§

InvalidPassword(Cow<'static, str>)

The provided password did not meet a length / encoding precondition.

§

InvalidSalt(Cow<'static, str>)

The supplied salt could not be decoded or was the wrong shape.

§

Hashing(HashingError)

The underlying primitive (Argon2 / bcrypt / scrypt / PBKDF2) reported a failure. The source error is preserved for structured downcasting.

§

Verification(Cow<'static, str>)

Verification failed for an internal reason (not a wrong password — that returns crate::Outcome::Invalid). For example, the stored hash was corrupt.

§

InvalidPolicy(Cow<'static, str>)

The crate::policy::PolicyBuilder could not produce a valid crate::Policy — typically because a required field was missing.

§

Decode(DecodeError)

Generic I/O or codec error (base64 decode, UTF-8, JSON).

§

Pepper(Cow<'static, str>)

Optional pepper provider (KMS / HSM) reported a failure.

impl Error

Source

pub fn hashing( kind: HashingErrorKind, detail: impl Into<Cow<'static, str>>, ) -> Self

Constructs an Error::Hashing for the named primitive with the supplied detail. The detail accepts anything Into<Cow<'static, str>> so callers can pass string literals (zero-alloc) or owned Strings.

§Examples
use hsh::error::{Error, HashingErrorKind};

let err = Error::hashing(HashingErrorKind::Argon2, "memory cost too low");
assert!(err.to_string().contains("argon2"));
