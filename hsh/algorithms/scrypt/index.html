Scrypt with configurable parameters (default = OWASP-2025 minimum). Scrypt wrapper with configurable parameters.

Default parameters follow the OWASP Password Storage Cheat Sheet 2025 minimum recommendation: N = 2^17 (131 072), r = 8, p = 1, dkLen = 64. The previous default of N = 2^14 is below the OWASP minimum and should not be used in new deployments.
